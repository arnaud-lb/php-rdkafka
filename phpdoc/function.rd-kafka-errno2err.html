rd_kafka_errno2err

(PECL rdkafka >= 0.9.1)

rd_kafka_errno2err — Convert the system errno to a kafka error code

Warning

This function has been DEPRECATED. Relying on this feature is highly discouraged.

Description

rd_kafka_errno2err ( integer $errnox ) : integer

Converts the system errno errnox to a kafka error code.

Parameters

errnox (integer)

A system errno

Return Values

Returns a kafka error code as an integer.
