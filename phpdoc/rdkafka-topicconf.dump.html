RdKafka\TopicConf::dump

(PECL rdkafka >= 0.9.1)

RdKafka\TopicConf::dump — Dump the configuration

Description

public RdKafka\TopicConf::dump ( void ) : void

Dumps the configuration properties and values to an array.

Parameters

This function has no parameters.

Return Values

Returns an array with configuration properties as keys, and configuration values as values.
