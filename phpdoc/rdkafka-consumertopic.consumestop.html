RdKafka\ConsumerTopic::consumeStop

(PECL rdkafka >= 0.9.1)

RdKafka\ConsumerTopic::consumeStop — Stop consuming messages from partition

Description

public void RdKafka\ConsumerTopic::consumeStop ( integer $partition )

Stop consuming messages for partition, purging all messages currently in the local queue.

Parameters

partition

Partition ID

Return Values

Returns no value.
